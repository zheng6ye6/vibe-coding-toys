# YSTI · 原神性格试炼

类似 MBTI 的二十道单选题，将四维度倾向与《原神》高人气角色比对，输出同调角色、星芒图、AI 场景插画与可分享的卡片链接。

## 启动（推荐）

分享功能（微信/QQ 卡片预览）需要服务端渲染 Open Graph 标签：

```bash
cd C:\Users\66478\projects\ysti
npm install
npm start
```

浏览器打开 **http://localhost:3456**

## 功能

- **20 题性格试炼** → 四维度加权匹配 18 位高人气角色（芙宁娜、那维莱特、玛薇卡、阿蕾奇诺等）
- **性格星芒图** → E/I/S/N/T/F/J/P 八极权重可视化
- **AI 场景重现** → 竖版 832×1248 插画（Pollinations，需联网）
- **公众号式分享** → 复制 `/s/xxx` 链接；好友聊天窗口显示结果卡片图，**点击进入 YSTI 首页**开始测试

## 文件

| 文件 | 说明 |
|------|------|
| `server.js` | 静态站 + `/s/:token` 分享落地页（OG 图 + 跳转首页） |
| `data.js` | 题库与角色 |
| `radar.js` | 星芒图 |
| `decor.js` | 元素意象装饰 |
| `scene.js` / `share.js` | 生图与分享卡片 |
