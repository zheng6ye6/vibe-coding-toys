# YSTI · 原神性格试炼

类似 MBTI 的二十道单选题性格测试，将答题者的四维度倾向与《原神》角色档案比对，输出最接近的提瓦特同调角色。

## 本地运行

在项目目录下启动静态服务器，例如：

```bash
npx --yes serve .
```

浏览器打开提示的地址（一般为 http://localhost:3000）即可。

也可直接用浏览器打开 `index.html`（部分环境对本地脚本有限制，推荐用上面的方式）。

## 文件说明

- `index.html` — 欢迎页 / 答题 / 结果三屏结构
- `data.js` — 二十道题目与角色向量数据
- `app.js` — 答题流程与匹配算法
- `styles.css` — 提瓦特风格界面

## 场景 AI 插画与分享

- 结果页会根据你**得分最高的 3 道题**与匹配角色，自动生成英文提示词，并调用 [Pollinations](https://pollinations.ai) 免费生图接口绘制「试炼场景重现」插画（需联网，首次约 10–30 秒）。
- **分享（图片链接）**：合成带 YSTI 标题的结果卡片并尝试上传至图床；成功后复制/分享的是**可直接在浏览器打开的 PNG 图片 URL**。若上传失败，则回退为场景 AI 图直链。
- **复制图片直链**：始终可复制当前场景图 URL（打开即为图片）。
- **结果页链接**：`share.html?r=…` 含答题数据，好友打开可看到相同角色与场景说明，并加载同 seed 的场景图。

分享页纯图片模式：`share.html?img=https://…`
